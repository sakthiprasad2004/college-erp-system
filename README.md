# College ERP System

A role-based College ERP system built using Spring Boot, React (Vite), and MySQL.

## Roles & Permissions

### Admin
- Create and manage students, faculty, and parents
- Assign students to parents
- Assign courses to faculty

### Faculty
- View assigned courses
- Mark student attendance
- Enter internal marks

### Student
- View attendance
- View internal marks

### Parent
- View child’s attendance
- View child’s internal marks

## Tech Stack
- Backend: Spring Boot, Spring Security, JPA, Hibernate
- Frontend: React (Vite)
- Database: MySQL
- Authentication: JWT
